# What is Math library in python?
The **math library** in Python is a built-in module that provides ready-made functions for doing mathematical calculations like square roots, powers, trigonometry, constants, etc.

## Common Python Math Libraries
- math – Basic math functions (sqrt, pow, sin, pi, etc.)

- statistics – Mean, median, mode, standard deviation

- random – Random numbers and selections

- numpy – Advanced numerical computing and arrays

- scipy – Scientific and technical computations

- sympy – Symbolic mathematics (algebra, calculus)
